== Description ==

With the private content extension you can provide logged in users with content that only they can access and view.

= Features of the plugin include: =

* Creates a private content area in wp-admin for each user.
* Site admin can add content in wp-admin.
* [um_private_content] shortcode allows for private content to be viewed anywhere on site
* Users can view their unique private content on their profile page in the private content tab
* Only users who have had private content added by admin will see the private content tab
* Private content tab can be disabled for profiles
* Admin can choose name of profile tab and select an icon from settings

Read about all of the plugin's features at [Ultimate Member - Private Content](https://ultimatemember.com/extensions/private-content/)

== Installation ==

1. Activate the plugin
2. That's it. Go to Ultimate Member > Settings > Extensions > Private Content to customize plugin options
3. For more details, please visit the official [Documentation](https://docs.ultimatemember.com/article/1497-private-content-setup) page.
