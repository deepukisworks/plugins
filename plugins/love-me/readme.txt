== Description ==

Display a very simple and customisable like button for your posts or any custom post type.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/love-me` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate it through the 'Plugins' screen in WordPress
