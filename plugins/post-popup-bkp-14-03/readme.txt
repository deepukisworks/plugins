**WP Post Popup plugin** for easy to show popup in popup in [wordpress](https://en.wikipedia.org/wiki/WordPress).

**Post in popup wordpress** allow to mobile **display post in popup wordpress**. visitor can be open **post in popup in wordpress**
so easy to read all post descritioin and title in popup


> **Plugin Features**
>
> * Quick view button in customize color, font size, background
> * Hide/show Option in author, category, date
> * Popup Body in customize color, font size, background

If any one visitor in fast for **show post content in popup wordpress** to get description as much as faster to **wordpress open post content in popup**.

<h4>Use With Shortcode Post Popup</h4>
For shortcode using **[post_popup id="93"]** need to use that code. just need to pass Post Id.

<h4>Using Function in Post Loop</h4>
If default button not coming in category page than you need to use below function:

    ♦ <?php
    ♦ 	oc_popup_post_button();
    ♦ ?>
